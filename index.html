    Exercício 4: Bhaskara

    Crie 3 inputs para a, b e c. Ao clicar no botão, calcule as raízes da equação usando Bhaskara. Mostre o resultado dentro de uma tag h2 com: cor azul se tiver uma raiz real, verde escuro se tiver duas raízes reais, e vermelho se não houver raízes reais (junto com a frase: "Não existem raízes reais").

    Observações para testar:
    - Para duas raízes reais: a = 1, b = -3, c = 2 → (x₁ = 2, x₂ = 1)
    - Para uma raiz real (delta = 0): a = 1, b = 2, c = 1 → (x₁ = x₂ = -1)
    - Para nenhuma raiz real (delta < 0): a = 1, b = 2, c = 3 → "Não existem raízes reais"
